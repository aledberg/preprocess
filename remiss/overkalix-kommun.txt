             Remissvar av Överkalix kommun Diarienummer S2024/00085
             Remiss av betänkande. SOU 2023:62: Narkotikautredningen - Vi kan bättre! Kunskapsbaserad
             narkotikapolitik med liv och hälsa i fokus

             Sammanfattning

             Hög dödlighet vid skadligt bruk eller beroende av narkotika
             Överkalix Kommun ställer sig positiva till utredningens förslag att ta fram ett program för att
             minska dödsfall till följd av narkotika- och läkemedelsförgiftningar.

             Förebyggande insatser
             Överkalix Kommun ställer sig positiva till utredningens förslag gällande förebyggande insatser
             för att skapa bättre förutsättningar för att förebygga narkotikaanvändning.
             Kommunen anser att det är särskilt viktigt:
                   •     att fortsätta pågående satsningar för ökat föräldrastöd. Intressant vore om det gick att
                         utveckla evidensprövade program så att de gick att använda på distans med e-lösningar.
                   •     att föräldrastödet kompletteras med information om hur föräldrar kan kommunicera
                         om narkotika med sina barn.
                   •     att genom Folkhälsomyndigheten tillhandahålla en kunskapsbas som grund för olika
                         aktörer
                   •     att stärka det narkotikaförebyggande arbetet genom att anta ett program för att
                         förebygga narkotikaanvändning hos barn och unga.
                   •     att länsstyrelserna fortsatt kan förfoga över resurser för att kunna stödja kommunerna i
                         genomförande av ANDTS arbetet. Ett Regionalt stöd bör ha stor betydelse för att
                         framförallt de mindre kommunerna ska kunna upprätthålla ett långsiktigt förebyggande
                         arbete.

             Vård- och stödinsatser vid skadligt bruk eller beroende
             Överkalix Kommun ställer sig positiva till utredningens förslag att utveckla kunskap, stöd och
             uppföljning i arbetet med att utveckla vård- och stödinsatser vid skadligt bruk eller beroende.

             Läkemedelsassisterad behandling vid opioidberoende (LARO)
             Överkalix Kommun ställer sig positiva till utredningens förslag för att utveckla arbetet med
             Läkemedelsassisterad behandling vid opioidberoende (LARO).

Socialtjänstens insatser
Överkalix Kommun ställer sig positiv till förslaget att kartlägga olika typer av boendeformer
utifrån olika lagstiftningar för personer med skadligt bruk eller beroende, och också utreda
behovet av vägledning inom området för att skapa mer enhetlig tillämpning av regelverket i
Sverige.

Statens institutionsstyrelse
Överkalix Kommun ställer sig positiv till förslaget att en så kallad individuell plan (SIP) även ska
upprättas när en person vårdas med stöd av LVM och Statens institutionsstyrelse (SiS) bedömer
att det behövs för att personen ska få sina behov tillgodosedda.

Kriminalvården
Överkalix Kommun ställer sig positiv till utredningens förslag inom kriminalvårdens område.

Den rättsvårdande processen
Överkalix Kommun ställer sig positiv till utredningens förslag gällande den rättsvårdande
processen.

Skadereducerande insatser
Överkalix Kommun ställer sig positiva till utredningens förslag om att se över hur ett
skadereducerande arbete kan stärkas.

Sprututbyten och tillgång till sterila sprutor och kanyler
Överkalix Kommun ställer sig positiva till utredningens förslag att ändra lagstiftningen för att
möjliggöra att utveckla de skadereducerande insatserna vid sprututbyten.

Information om narkotika till användare
Överkalix Kommun ställer sig positiva till utredningens förslag om att ge Folkhälsomyndigheten i
uppdrag att tillsammans med civilsamhället och målgruppen utreda bästa sätt för att ta fram
och sprida upplysningar om olika substanser och värderingsfri information om risker till
personer som använder narkotika.

Anhörigas situation positiva till förslaget
Överkalix Kommun ställer sig positiva till utredningens förslag för att öka kunskapen om
anhöriga till personer med skadligt bruk eller beroende och/eller samsjuklighet, samt förslag för
att tydliggöra ansvar för stöd till anhöriga till en person med skadligt bruk eller beroende av
narkotika.

Insatser för att öka kunskap och kompetens
Överkalix Kommun ställer sig positiva till utredningens förslag på insatser för att öka kunskap om
vård, stöd och skadereducering vid skadligt bruk eller beroende.
